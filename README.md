# Tributary Pro v2.0
Professional structural engineering web application for 2-way reinforced concrete slab buildings.
## Features
- Two-way slab tributary distribution (45° rule)
- Proper load path: Slab → Beams → Columns
- Beam tributary widths with triangular/trapezoidal loads
- Column reactions from beam end moments & shears
- Multi-floor stacking
- Interactive grid visualization
- Foundation load output
## Scope (v2.0)
✅ Regular rectangular 2-way slab systems
✅ Uniform distributed loads (dead + live)
✅ Factored load calculations (NSCP/ACI)
## Coming in Future Versions

- v2.2: One-way slab option
